# Présentation du projet

Nous sommes un groupe d'étudiant de l'IUT Nord-Franche-comté qui, dans le cadre d'une situation d'évaluation et d'apprentissage (SAÉ),doit créer un site web présentant une entreprise française du secteur informatique.
Nous avons choisis l'entreprise SFR car c'est une société française avec un parcours intéressant, qui a concurrencer France-télécom à ses débuts et qui a sû être une alternative sérieuse aux concurrents. Dans le cadre de cette SAÉ, SFR(fictif), nous demande de créer un site web pour présenter leur entreprise. Nous devons donc élaborer des questionnaires client afin de comprendre la demande du client pour pouvoir lui créer un site web qui coresspond aux attentes.
